void display_7_segment ( float  value,
uint8_t  no_of_decimals 
)

Updates the display's content.

Updated the display to show the given value with a given number of fraction decimals. For example: display_7_segment(12.389,2) will display 12.39

Parameters
[in]valuethe floating point number to be displayed
[in]no_of_decimalsthe number of fraction decimals to be shown [0...3]
Note
If no_of_decimals > 3 then 3 fraction decimals will be displayed
The displayed value is rounded

◆ display_7_segment_create()

void display_7_segment_create ( )

Constructor function for 7-segment display driver.

Sets up the PB0-2 and PF0-3 pins as outputs, set up the SPI if its use defined and sets up TIM0 to 400Hz for display character refresh.

Note
Must be called once before any display functions must be used!
To use the interrupts sei() must be called after this function!
