void rc_servo ( int8_t  percent)

Sets the servo to a position.

Percentage translates the following way: -100 -> 1ms pulse 0 -> 1.5ms pulse 100 -> 2ms pulse

Values of <-100 and >100 are ignored.

[in]percentposition percentage

void rc_servo_create ( )

Constructor function for rc_servo driver.

Initialize the pin PL4 as output and sets the TIMER5 with 20ms period. It sets the OCRB as output with 1.5 ms.

Note
Must be called once before the rc_servo functions must be used!
